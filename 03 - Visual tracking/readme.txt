Hay 2 metodos :

Si utilizas un video -> modificar esos parametros :

         - using_video -> true
         - video_path -> camino del video (ejemplo : 'lapiz.avi')
         - sequence_dir_path -> dar un nombre al directorio donde estaran las imagenes (ejemplo : 'images')


Si utilizas una sequencia de imagenes -> modificar esos parametros :

-> Las imagenes utilizadas tienen que tener como nombre un numero (orden) y .jpg

         - using_video -> false
         - video_path -> ''
         - sequence_dir_path -> nombre del directorio donde estaran las imagenes (ejemplo : 'images')
